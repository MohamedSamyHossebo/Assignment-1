# Assignment-1 Repository README

## Overview

This repository contains all parts of the assigned tasks, organized clearly for easy navigation and execution.

## Repository Structure

* **Part 1 — Problem Solving (`index.js`)**
  Contains JavaScript problem‑solving questions implemented directly in `index.js`.

* **Part 2 — Essay Questions (`essay.md`)**
  Includes all required written explanations for the essay section.

* **Part 3 — Bonus Problem Solving (`bonus.js`)**
  Solutions for the bonus challenges based on LeetCode‑style problems.

## How to Run the Code

Make sure you have **Node.js** installed.

### Run any file with live reload (watch mode):

```bash
node --watch filename.js
```

Examples:

```bash
node --watch index.js
node --watch bonus.js
```

This will automatically re‑run the file whenever you save changes.

## Notes

* All code files use plain JavaScript.
* No external libraries are required.
* Essay answers are located in `essay.md` for clean separation.

You're ready to start testing and reviewing all assignment parts!
